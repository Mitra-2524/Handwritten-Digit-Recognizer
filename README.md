# Handwritten Digit Recognizer

## Description
This project implements a Convolutional Neural Network (CNN) to recognize handwritten
digits from 0 to 9 using the MNIST dataset.

## Dataset
MNIST Handwritten Digits Dataset consisting of 28x28 grayscale images.

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy

## Model
Convolutional Neural Network (CNN)

## Output
The model achieves approximately 99% accuracy on test data.

## How to Run
1. Install required libraries:
   pip install tensorflow numpy
2. Run the program:
   python digit_recognizer.py
